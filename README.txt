نظام إدارة منتجع أضواء الشرق

تشغيل سريع:
1) افتح index.html على الكمبيوتر.
2) على الجوال: ارفع المجلد إلى استضافة HTTPS ثابتة، ثم افتح الرابط واختر "إضافة إلى الشاشة الرئيسية".
3) البيانات تحفظ في نفس الجهاز والمتصفح، لذلك استخدم زر "تصدير نسخة احتياطية" بانتظام.

المزايا:
- لوحة رئيسية وحساب المحصل والمتبقي.
- إدارة الحجوزات ومنع تداخل التاريخ.
- التقويم.
- المصروفات وصافي الدخل.
- زر واتساب.
- إدخال صوتي أساسي.
- تصدير JSON وCSV.
- يعمل دون خادم وقاعدة بيانات خارجية.
